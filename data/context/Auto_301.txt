新锐志等11款新车将发布 E3馆看车指南
网上车市4月20日报道
以“畅想绿色未来”为主题的2010北京国际车展，将于4月25日-5月2日在北京中国国际展览中心隆重举行，届时国内外汽车品牌将有众多的新车、概念车首发，而在车展开始日期日益临近之际，各个品牌的参展车辆也陆续浮出水面。为了能够让您更好的了解本届北京车展，网上车市将各个场馆即将发布的车型进行汇总，以便对您的参观有所帮助，下面就让我们一起来了解一下E3馆有哪些新车发布。
 
 
在E3馆中，只有5家汽车厂商在此参展，而丰田和雷克萨斯则占据了半壁江山，旗下的 新款锐志、新普锐斯、和MPV车型——Alphard也将与广大消费者见面。三菱以进口方式引入的跨界车RVR上市销售，后期还将国产。江淮汽车在本次车展上推出了多款新车，愿景、悦悦、和悦自动款、瑞鹰2010款、和悦RS自动款和新宾悦自动款都将于国内消费者见面。

在国内市场上，一汽丰田旗下的锐志车型一直是运动型中级车的典范之作，虽然外观设计比较低调，但是凭借着后驱设计与2.5L V6发动机等出色的运动基因赢得了广大消费者的认可。在即将举行了2010北京国际车展上，一汽丰田将带来进过全新设计的新款锐志。目前这款车已经在日本本土上市销售，那么我们就根据这款车的海外图片来抢先了解一下这款新车。
 
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

这款车型搭载了2.5L 和3.5L发动机（国内老款锐志为2.5L和3.0L），并分为普通版、高端版和运动版三个版本一共8款车型。新一代锐志外观上已经很难再找到现款锐志的设计元素，倒梯形前格栅和波浪型前大灯都使这款车显得更加凌厉和富有运动感，并且与雷克萨斯车型颇有几分相似。
 
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

新锐志的长×宽×高分别为1730mm×1795mm×1435mm，轴距达到了2850mm。与目前销售的锐志车型相比，新款锐志车身长度和车身高度有所缩短，而但是宽度明显加大，这样的设计使这款车在行驶时的稳定性更出色。2850mm的轴距长度则完全能够满足消费者对于空间的需求。
 
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

外观上，普通版的外观更加具有运动感，上下进气格栅更加张扬，而豪华版则要中庸得多，更加类似豪华车型的沉稳设计。另外，普通版车型将采用17寸轮合金轮毂，而豪华版与运动版将匹配更加动感的18寸轮合金轮毂，并且将采用235/45高扁平率的宽胎。内饰上，运动版车型将采用更加动感的全黑内饰设计。
 
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

与老款锐志相比，新锐志的尾部设计一改老款圆润的设计，变得更加大气和富有力量感。与车头厚实和饱满的特点相一致，这款新车的尾部设计同样采用了更加硬朗的线条。尾灯设计有一点雪佛兰科鲁兹的味道；而现款锐志标志性的内嵌式方形双尾排气管被普通的圆形排气管所取代，运动的味道没那么浓烈。
