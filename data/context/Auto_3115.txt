三菱君阁升级版22日上市 13.68万元起售
版权声明：本文版权为网易汽车所有，转载请注明出处。
 
网易汽车5月19日报道
东南汽车宣布，经过升级的新款三菱君阁将在5月22日上市，售价仍为13.68万元起。
 
| 
| 
| 
| 
| 
据介绍，此次升级针对安全、舒适、座椅多功能设计进一步强化，一二排或二三排可组合平躺；二排座椅滑动、4/6分离并可折叠前翻，三排五五侧翻，实现最大储物空间1878L。
 
三菱君阁采用以帕杰罗高刚性底盘为基础而设计的新底盘，前双A 臂独立悬架和多连杆螺旋弹簧后悬架配备，即使是崎岖不平的路段也能保持平稳的行驶；搭载三菱2.0L MIVEC发动机，具备可变气门正时系统，最大功率高达125kw/6400rpm，实现欧IV排放标准。
 
在内饰上，三菱君阁拥有海湾式中控台设计、7"TFT触摸屏DVD、车载蓝牙免提功能，轴距达到2720mm。此外，经过升级的三菱君阁全系标配前后排空调、双安全气囊、四门防撞钢梁以及可溃缩式方向机柱等。
