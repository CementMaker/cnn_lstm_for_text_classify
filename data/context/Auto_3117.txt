路虎Freelander／LR2改款车路试谍照曝光（图）
版权声明：本文版权为网易汽车所有，转载请注明出处。
网易汽车5月20日报道
日前，国外媒体的谍照摄影师拍到了一辆正在进行路试的轻微伪装的改款LR2原型车，新车似乎拥有了一个经过改动的前保险杠面板。预计新款路虎Freelander／LR2改款车可能会在今年晚些时候开幕的2010年巴黎车展上发布亮相，让我们共同期待。

路虎或许正在加紧研发其全新的LRX车型，但是这并不意味着路虎系列的其他车型就会不受重视。新版的揽胜与揽胜Sport车型将于2010年推出，LR3的替代车型LR4同样也将在2010年推出，而且如今看来，路虎还将推出一款小改款的LR2（或Freelander）以迎接新的车型年。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

从这些谍照看，似乎路虎将通过在LR2车头的中部再增加一个进气口使得改款LR2的前保险杠拥有一个全新的面貌，另外前雾灯的位置也将改变，预计会被安装在车头更低的位置上。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

除此之外，新款LR2似乎不会有其他太多的改动，不过预计2011款LR2的内饰也会得到一些改动与调整，希望新款LR2能够从路虎其他车型富有革新性（而且获得了很大改进）的仪表板设计上获得一些设计灵感。
