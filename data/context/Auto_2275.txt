奔驰E200/E260同为1.8T引擎 差价6万如何选
网上车市5月7日报道 
奔驰旗下全新E级轿车在中国市场已经推出了3个产品系列，按照型号区分包含E200 CGI、E260CGI和E300。但是如果按照引擎来划分，那么事实上则只有1.8升和3.0升两大系列。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

其中E200 CGI和E260 CGI均使用了1.8升涡轮增压引擎，但是价格却相差了6万元，那么对于消费者来说究竟如何选择呢？我们不妨一起来找找看这两款车型有何不同。
1.动力部分
可以看到，在价格上E200和E260相差了6万元，但是两款车型的型号却完全不同。至少直观上的理解大多数人都会认为E260配备的引擎更大，更高端。那么在动力和排量上，是否真的如此呢？
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

事实上奔驰E200 CGI和E260CGI均使用了一款1.8升直列4缸涡轮增压引擎。从数据上看，两款引擎的汽缸工作容积均为1796毫升、缸径*行程也完全一致。只是在功率、扭矩输出上由于引擎的不同调教E260的引擎动力会更大，在动力上相比E200会更加丰沛一些。两款车型的百公里加速时间仅仅相差0.4秒，最高车速也都能够达到230公里/小时以上。所以如果您对动力没有太多苛刻的需求的话，E200K的动力是完全能够满足日常使用的。
2.外形方面的对比
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

E200与E260相差6万元的价格在外型上还是很容易区分的，首先两者的尾部标识是最为明显的，一个是E200一个是E260。其次两者的轮毂方面一个是双10辐一个是双5辐的设计，以及轮胎方面一个是205的宽度一个是225的宽度。事实上，上述差异都不会让驾驶者感觉到十分明显的区别。
3.内饰差异比较大
经过前面动力系统、外形方面的介绍，我们接下来再来看看E200和E260的内饰差异。首先两者的中控台上都配备了7寸显示屏，并且能够播放DVD。但是E200中控台上的DVD换碟机为单碟式，而E260为6碟的播放器，在容量上要不E200更多一些。其次是后排车门以及后窗的位置没有设计遮阳帘，这可能会对后排乘客的私密性和舒适性有些影响。还有E200的空调并非多区独立控制的自动空调，在车内木材的选用上也使用到了相对廉价的多疤胡桃木。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

总体来说，奔驰E200与E260的6万元差距全部都体现在了外形和内饰的配置上，动力调教的区别并不十分明显。因此如果您认为没有必要在豪华配置上花费过多，那么奔驰E200 CGI不足49万元的价格真的会是一个高性价比的选择。
