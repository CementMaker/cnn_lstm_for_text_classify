外形酷似新福克斯 现代全新伊兰特发布
网上车市5月6日报道
刚刚结束的北京车展云集了数十家厂商的近千款车型，其中许多厂商都将旗下的重量级车型带到了本次车展的现场。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

其中，在现代汽车的展台上我们如约地看到了iX35和VERNA，但是现代旗下的全新一代伊兰特却并未到场，而是出现在了5月上旬开幕的韩国釜山车展。下面我们就来一起仔细的了解一下这款全新的伊兰特吧。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

在看到全新一代悦动的外形时，首先会感觉到它的造型非常犀利，整体上感觉更像是一款硬朗的欧系轿车。直觉上看它的外形倒是和此次北京车展现场出现的福特全新一代福克斯有着几分神似。而福克斯和伊兰特以及悦动恰好也是在中国紧凑型家轿市场中最为畅销的两款车型，也许不久的将来我们就可以在国内同时看到这两款车了。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

与2011现代索纳塔、劳恩斯轿车、跑车以及雅科仕加长版车型一样，新款伊兰特车型也将在车身外观上发生重要改进。新款伊兰特车型在外观设计方面更具有跑车的车身风格。新车的引擎罩以及前保险杠也最为最新的设计。前车灯将更长、更具有棱角风格。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

车身的侧面线条相比现在北京现代国产的伊兰特悦动更加运动，一条从车头部分贯穿车身直达车尾的腰线提升至了车辆侧面60%以上的高度。并且这条腰线像是许多运动风格的车型一样采用前高后低的姿态看上去动感十足。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

2011现代伊兰特的详细规格配置目前还不得而知。不过预计现代新推出的Theta II燃油直喷四缸发动机将有可能配置在该车上，与之相匹配的将有可能为6速自动变速箱。此外还将可能包括一款1.6L涡轮增压四缸发动机。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

作为现代汽车最为重要的市场之一，现代汽车在中国市场的表现非常出色。北京现代旗下的伊兰特、伊兰特悦动一直以来都是同级别、同价位车型中的佼佼者，其中伊兰特悦动更是能够达到每个月2-3万台的销量。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

不过令我们感到有些不解的是，这款悦动的升级车型并未选择在早已成为国际A级车展盛会的北京国际车展上进行发布。这不能不算是一个不小的遗憾，但是我们相信有着中国市场的庞大购买力，这款全新车型早晚都会成为北京现代旗下的一款国产车型的。
