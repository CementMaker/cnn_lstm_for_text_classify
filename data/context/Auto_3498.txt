网易汽车5月26日报道 
梅甘娜CC
早在2006年就进口引进的方式登陆中国市场，但真正了解这款车的消费者并不多，但在法国本土，梅甘娜系列却深受大多数消费者喜爱。目前雷诺发布了2011款全新
梅甘娜CC

，比起老款沉闷的外形，新款更有活力。雷诺于今日公布了2011款梅甘娜CC的售价与参数。新款的梅甘娜CC预计最早将于7月9日抵达展厅。2011款提供两个版本DyanmiqueTomTom与GT，同时动力方面包括多款汽油与柴油发动机。

汽油发动机包括有1.4升的TCe最大马力为130匹，还有2.0升的TCe最大马力为180匹，2.0升CVT最大马力为140匹；柴油版方面包括有1.5升dCi最大马力为110匹，1.9升dCi最在马力为130匹以及2.0升dCi最大马力为160匹。2011款的新增装备包括有17英寸合金轮毂，蓝牙，全景电子天窗，全集成的CarminatTomTom导航系统等。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

如果选择的是汽油版，起售价为21595英镑（约合21.2万元），如果是柴油版起售价为23595英镑（约合23.1万元）。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

新车系共提供六款发动机。两款为GT型号，另外四款为另一型号，新增的EDC双离合自动变速器，不过仅供应于1.5升dCi这一款车型上，CO2排放仅为130克/公里，在同级敞篷车中属于最低的。GT型号所提供的两个车型均配备两款2.0升的发动机，保证强劲的动力。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

而外观方面也有改变，如在车外采用新的前保险杠，中央进气口位置稍有变化增添黑色装饰组件，后保险杠新增空气分流器。风挡框架及腰线位置采用暗色的镀铬包围，与门把手，后视镜和前雾灯的设计相呼应。高端型号上直接配备18英寸的合金轮毂并采用黑色的包围。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 |

旗舰型号在车内还增加GT风格的运动座椅，新的指针式转速表，抛光的碳纤维内门把手，空气出风口和带有GT标志的仪表板，还提供倒车雷达与电动折叠后视镜。可选装备方面包括有金属漆，双区空调，可加热前座椅，皮革内饰，双氙头灯，卫星导航，3D音响等，售价均在数百英镑内。
