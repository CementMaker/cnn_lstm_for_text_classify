网易汽车4月15日报道
随着中国汽车市场的持续增长，越来越多的车型被引入到国内。30万到50万元是个门槛，高级车型的序幕至此开始。
从这个区间开始，各大厂商“血拼”的已不再是价格，更多的是品牌以及品牌赋予的产品内涵。在4月23日的北京车展上，这些车型将全部亮相。
双门轿跑及敞篷轿跑
预计售价：50万元起
关注指数：★★★★★

全新BMW 3系双门轿跑车及敞篷轿跑车在前部、侧面和后部均进行了精心的设计改进，更新的发动机系列以及创新的设计特点使双门车型比以往更优雅、动感和高效。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

外形上，前部、侧面和后部的创新设计增强了全新BMW 3系双门轿跑车和敞篷轿跑车超凡的运动感，流畅而富有活力。氙气大灯带LED日间行车灯也运用在该车身上，双圆形大灯改进了轮廓线并采用了创新的照明技术，其顶部融入饰条，形成了宝马特有的前大灯造型。尾灯采用了宝马经典的夜间照明设计，全红色玻璃罩可在两排灯同时点亮时融入更加和谐、均衡的光线感。
动力方面，全新BMW 335i双门轿跑车和335i敞篷轿跑车将搭载新开发的直列6缸发动机进入市场，首次将宝马双涡管单涡轮增压发动机技术、高精度直喷系统和Valvetronic电子气门管理结合在一起。

作为宝马高效动力（BMW EfficientDynamics）的一部分，此次在欧洲市场推出的BMW 318i、BMW 320d和BMW 325d也都加入了BMW 3系双门轿跑车和敞篷轿跑车的可供产品阵容。其中，新的入门级车型BMW 318i双门跑车系列装备了2.0升4缸汽油发动机，采用高精度直喷系统以及稀薄燃烧技术。
