作为劳斯莱斯的旗舰车型，幻影不仅仅是因为它拥有5.834米的长度、1.632米的高度以及1.99米的宽度这样惊人的数据，更多的是它所散发尊贵与品质，直瀑式的镀铬进气格栅、宽厚稳重的前脸以及银天使立标，都是任何豪华车模仿不来的。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

车内多处复古的设计处处彰显了这个拥有百年历史的英国品牌的内涵与神韵，不仅看上去美观大方，而且实用性也很强，手感以及人机工程都处理的很到位。像独立的仪表、中央扶手、空调/出风口/主机等的面板，都是非常复古的设计。
劳斯莱斯幻影配用源自宝马的6.75升V12发动机，最大扭矩高达720牛米，最大功率为453马力。极限动力并不是它特别追求的，极佳的振动和噪音控制以及超乎任何人想象的平顺性，才是显示其绝对不同于普通轿车的地方。
