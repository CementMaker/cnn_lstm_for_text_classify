整合旗云 奇瑞“五大品牌”战略格局全面确立
版权声明：本文版权为网易汽车所有，转载请注明出处。
网易汽车5月24日报道 
27日，奇瑞汽车将正式公布“旗云品牌”相关细节。在此之前，网易汽车从奇瑞汽车内部获悉，按计划，奇瑞品牌下的QQ6平台的一款新车，以及A5和旗云的升级版将被纳入这一子品牌中。“旗云品牌”的整合，标志着奇瑞汽车销售有限公司旗下的QQ、A系、瑞虎、风云、旗云“五大品牌”发展战略格局全面确立。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

奇瑞要打造的全新“旗云”定位是“奇瑞品牌旗下值得信赖的大众化汽车品牌”。奇瑞内部人士进一步透露，随着旗云整合的完成，旗云123产品的推出，“旗云品牌”下一步将开始着手丰富产品系列的工作。旗云5、旗云6、旗云7等全新家用轿车将陆续上市。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

对于“旗云品牌”发展的规划，奇瑞汽车股份有限公司副总经理、奇瑞汽车销售有限公司总经理马徳骥表示，我们对旗云进行做好了三个阶段的规划：“第一阶段，也就是2010年，旗云要‘成为经济型轿车的重要品牌’；第二阶段，到2013年，成为经济型轿车的主力品牌；第三阶段到2015年，成为经济型轿车的领导品牌。”
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

2009年，奇瑞汽车股份有限公司发布了奇瑞、瑞麒、威麟、开瑞四大品牌，形成了多条营销渠道支撑的“大奇瑞”多品牌架构。此次整合“旗云品牌”，奇瑞认为更是“大势所趋”。马德骥表示，“旗云上市10年以来，经济耐用的品牌内涵深入人心，已经形成了鲜明的品牌文化，吸引着众多消费者。与此同时，汽车市场也形成了一类大众化、经济实惠、皮实耐用的家用轿车细分市场。所以，现阶段整合旗云品牌，强化旗云品牌，无论是对于市场还是对于旗云，都是正确的选择。”
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

奇瑞旗云2001年上市之前，中国汽车业还处于高价时代。彼时，合资企业生产的汽车占领绝大部分市场份额。对于国内消费者来说，想要拥有一辆私家车，必须承受较高的购车成本。这种情况下，旗云的上市，不仅终结了高价汽车时代，还将“ABS+EBD”作为标准配置引入9万区间，促使这项当时的“高端安全配置”成为国内A级车的主流标配。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

发展6年以来，旗云行销全球七十余个国家和地区，在国内销售总量突破80万辆，目前无论在国内市场还是在国际市场，旗云都几乎占据了奇瑞汽车销量的半壁江山。源于此，奇瑞决定以“旗云”作为整合后的品牌名称进行继续打造。
当今车市，车市竞争日趋白热化，尤其10万区间市场成为诸多厂家的必争之地，各汽车品牌均呈现快速发展的态势。一场大规模“洗牌”正在悄然进行，谁将最终称雄市场，让我们拭目以待。
