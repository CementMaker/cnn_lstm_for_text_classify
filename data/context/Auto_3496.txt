2012款宝马6系敞篷版 谍照现身图（图）
版权声明：本文版权为网易汽车所有，转载请注明出处。
网易汽车5月26日报道 
新的2012款宝马跑车版与敞篷版都要明年才能正式上市，因为现行一代两年前才刚刚完成一次小改，这已经是最快的换代速度了。现行一代的宝马6系销售非常成功，每年能保持在2万部的销量，对于这个级别来说是很优异的成绩。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

不过我们发现新一代6系的原型测试车谍照已经在欧洲现身了。在慕尼黑和纽博格林都曾留下过它的身影，这次我们得到的是敞篷版的谍照，相比之前我们看到的跑车版的谍照，这次许多细节更加清晰了，从后侧车窗处可以明显地看出两个车型的不同，顶部曲线也有明显的差别，跑车版的后车窗比敞篷版更略宽一些。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

这也表明宝马在6系上的处理是不同于Z4车系的，Z4中是只提供一套折叠硬顶，但是在宝马6系上是提供了传统的软顶。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

宝马不打算太多地影响到下代车型的销量，因而新车的设计者将没有多大的发挥空间，谍照显示新车外形更加修长，车体更宽，整体的运动感更为强烈。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

在车内部分，新6系有众多的豪华部件，比如8速自动变速器带拨片换挡功能，现行的发动机阵线有望保持不变，不过顶级的M6版本预计不会有变化，因为宝马可能计划推出新的高端跑车以替代原来M6的位置。
