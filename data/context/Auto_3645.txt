2012款现代雅绅特谍照曝光 明年中期上市
版权声明：本文版权为网易汽车所有，转载请注明出处。
网易汽车5月27日报道
全新2011款现代索纳塔已经上市，而新一代伊兰特也刚刚发布亮相，如今，人们的注意力开始转向现代的入门级车型雅绅特。日前，国外媒体的谍照摄影师拍到了正在德国进行测试的新一代雅绅特原型车。据悉，全新一代2012款现代雅绅特将于明年中期上市销售，将再一次推出掀背款和轿车款两种车身式样。

如同这些最新的谍照所显示出来的，新款雅绅特将采用与索纳塔和即将上市的伊兰特一样的流畅设计、陡然下降的前格栅以及逐渐变细的前灯。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

内部代号RB的新一代2012款现代雅绅特将基于与海外市场销售的i20紧凑款车型相同的平台设计。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

最可能的发动机选择将会是一款最大动力输出为140马力的1.6升四缸发动机。虽然这款发动机或许会有点缺乏力量，不过需要指出的是，与这款发动机相匹配的很可能会是现代的新款6速自动变速箱，将能够获得高达40 mpg的公路燃油经济性。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

此外，新款雅绅特还将提供一款标配的6速手动变速箱。现代公司还在考虑推出一款混合动力版的雅绅特车型，不过这款混合动力版车型至少还需要几年的时候才会发布上市。
