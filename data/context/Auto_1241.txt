敞出的动感 图文详解奔驰新E级敞篷版
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

要说车头造型算是好看，那么全新E级敞篷的侧身设计便算得上精彩了。主线条从保险杠探出，连续地向后延伸，强调了动感的契形车身。而令人为之拍手叫绝的是其后翼子板处腰线的设计。
另一条腰线由后轮处弯起延伸至尾灯，将侧面与尾部连接起来。而将车辆的车顶敞开后，它所散发出的那种运动、时尚气息完全可以彻底征服每一个看到它的人。双门的设计是每一款敞篷车型所惯用的，奔驰E级敞篷也不能例外，不过这也让其在乘坐实用性上有着一定的折扣，但对于一款敞篷车而言，它的实用性往往不是最为重要的。
尾部造型上，箭头型的LED尾灯比旧款来得更大，更加醒目。整个尾部层次分明，勾勒得当，很为成熟的设计，但是个人认为尾部造型相对于其他部分稍显平庸，作为一款敞篷车而言，它的尾部设计不够动感，还是较多的保持了奔驰E级商务的特性。
