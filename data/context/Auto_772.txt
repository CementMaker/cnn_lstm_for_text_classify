不变的"加长效应" 图解新一代奥迪A8L
版权声明：本文版权为网易汽车所有，转载请注明出处。
 
网易汽车4月24日报道
 在刚刚开幕的北京车展上，德系豪华车三剑客的三款首发车型（奥迪A8L、宝马长轴距版5系、奔驰长轴距版E级）尤为引入注目。作为奥迪旗舰车型，A8L不仅加长了130mm，而且搭载了新的W12引擎，无论外形还是性能都较旧款车型有较大的改进。不妨一起了解一下这款即将进入中国的车型。而这次三款车型的推出也代表着“加长”这个一直以来被持续争论的话题在国内还是很有市场卖点的。

 | 
	    
 |

从前脸来看，A8L较A8并无明显区别，巨型的一体式矩形进气格栅、飞翼式大灯颠覆了旧款A8沉闷的外形设计，显得更强大、更富有力量感。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

 
W12版本如期加入A8L的阵营，如同S8，W12版本在前进气格栅样式等方面有所改变，雾灯周围也被镶上镀铬饰条以彰显W12的旗舰身份。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

 
与老款A8正统呆板的大灯相比，新A8的前大灯“L”型的延展性设计显得格外立体。奥迪也舍得在车灯上撒下重金：22颗白光LED灯以及22颗黄光LED灯构成了强势的LED光源。其中白色LED负责日间行车灯功能，而黄光灯则充当转向灯。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

 
全新奥迪A8L的车身长度达到5267毫米，较普通版增加了130毫米，轴距达到3122毫米。从图片来看，B柱之前，A8L与A8并无二异，所加长的130mm轴距全部用于后排。奥迪在长轴距版本车型方面已有丰富经验，从A8L来看，车顶曲线十分流畅，甚至要比常规版本多几分优雅气质。
借用奥迪首席设计师Stefan Sielaff的诠释：第一代A8的设计很教条，第二代A8则凸显的是优雅和飘逸，而全新第三代奥迪A8将会为世人展示出一个更强大、更精确清晰的外形。A8L更将这种精神演绎得精彩十分，侧身看去，线条的运用更为明显，尤其在腰线的部分，将车身的流畅感与运动气息完美的结合。
