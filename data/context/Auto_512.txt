[探馆]心动感觉依然 再遇世爵C8 AILERON
版权声明：本文版权为网易汽车所有，转载请注明出处。
网易汽车4月22日报道
距本届北京车展开幕还有1天，网易汽车探馆团队在展馆现场抢先拍摄到了世爵C8 AILERON的图片。虽然在车展前的赏车会上已经有过近距离接触，但再次看到世爵C8 AILERON的此刻依然艳羡不已。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

C8 Aileron对于中国观众并不陌生，该车曾亮相于08年的北京车展，而此次展出的C8 Aileron为第二代升级产品，才从刚结束的日内瓦车展空运至京。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

从外观来看，新的世爵C8 Aileron仍延续旧款的车体造型，流线且富有圆润感的车身似乎永不过时。银色网状跑道形进气格栅努力呼吸每一口空气，即便两端圆形的雾灯也非密封，雾灯只占据圆孔最中央位置，新鲜的空气将从雾灯四周涌进，为刹车系统带走热量。大灯造型如一把利刀，细长锋利，而且世爵还为它画上潮流的眼线，一排LED日间行车灯。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

由于发动机采用后中置布局，车头采用大倾角设计，多处进气口设计除实际用途外，还具有强烈的视觉冲击力。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

侧面是世爵C8 Aileron最为柔美的一面，低矮的车身、流畅的线条没有半丝停顿，由车头延伸至尾部，顺延车顶下滑的曲线，尾部稍稍向下收住，并翘起一块灵巧的扰流器，如此熟悉的线条，对，这线条来自07年亮相的全球限量24辆的C12 Zagato！如同其他世爵跑车，C8 Aileron同样采用剪刀门设计，门与车身之间由坚固的轻质铝合金单点铰链连接。也许是传统的把手开关过于累赘，开启剪刀门的机关变成一个细小的圆形电控按钮，位于后视镜下方。
