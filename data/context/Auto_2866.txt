既然这两款车都定位在运动性车型上，它们在驾控方面一定都有着不错的表现，这次对比试驾正好让我们可以更好的感受到二者之间孰优孰劣。
》更多睿翼/思铂睿实拍图片 | 
|
 
|
 
|
 
|《
 
首先我驾驶的是睿翼轿跑，这次对比试驾中睿翼轿跑使用的车辆是2.5升车型，从动力的表现上来看，其搭载的这台来自马自达的拥有S-VT、VIS、TSCV等技术的直列四缸自然吸气发动机还是较为不错的，124千瓦/6550转的最大功率和226牛米/4000转的峰值扭矩，再加上力度较轻的油门，在日常驾驶中你绝对不会感觉到它的“肉感”。
》更多睿翼/思铂睿实拍图片 | 
|
 
|
 
|
 
|《
 
但与之相比思铂睿的发动机参数表现上则会更好一些，本田独有的i-Vtec技术让其搭载的2.4升发动机拥有着133千瓦/6500转的最大功率略高于睿翼轿跑车型，而225牛米/4300转的峰值扭矩则与其相类似，这样看来在动力表现上似乎思铂睿应该更好，但事实却不尽然。由于睿翼轿跑的车身自重要比思铂睿轻将近100千克，其最大扭矩输出点也要提前与思铂睿，所以当你驾驶车辆时你会感到睿翼轿跑的车辆提升更快，动力感更强。


》更多睿翼/思铂睿实拍图片 |

|《
 
无论是马自达还是本田，它们为其发动机安排的“搭档”均为5速自动变速箱，不过不同的是马自达的这款5速自动变速箱带有手动模式，而本田的这款则为纯粹的自动变速箱，并没有手动模式，但其却增加了运动模式。从换挡平顺性方面而言，它们都符合大部分消费者的要求，平稳驾驶时变速箱的换挡都较为平顺，但当你深踩油门时，变速箱都会稍显迟疑的进行降档工作，以提供更好的扭矩输出。而由于这两款车都配有拨片换挡功能，所以在试驾过程中我们也注意体会了一下这两款车的拨片情况，相比之下思铂睿所采用的“左减右加”的形式更加传统，而睿翼轿跑所采用的换挡拨片则更加具有观赏性，不过在使用时二者的换挡响应都较为良好。
