库存不断增加  国产近30款热销车型行情一览

 | 

继途锐以进口的形式进入国内市场以后，大众另外一款国产SUV途观也在近日落户上海大众，售价19.98万元起。在国产途观上市之前，这款新车的订单就已经累计到5万台左右，可见这款产品吸引了多少国内消费者。不过，面对如此庞大的订单数字，上海大众要如何消化掉？又要怎样去保证普通消费者的购车时间不会太久？
Q5
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

3月20日，国内高端品牌领跑者——奥迪，推出了备受瞩目的国产版Q5，售价区间为37.98万-53.98万。国产Q5的上市使高端紧凑级SUV市场竞争变得更为激烈，由于国产使车辆成本大大降低，在品质及配置不变的情况下售价也随之下降，相比进口版Q5整车性价比有大幅提升。
CR-V
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

新的CR-V在越野性能这方面，也有所改进。经过修改的适时四轮传动系统，可以侦测前轮打滑，并且将动力传导到后轮。比起之前所使用的系统，可以多传输20%的动力到后轮。虽然CR-V在设计之初，就不是以重度越野作为目标，不过在使用这套四轮传动系统之后，在对面轻度越野的路况，像是下雨天或者积雪时，CR-V也能够轻松应对。
RAV4
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

国产RAV4在外形设计上，进气口和前后保险杠更突出了越野的感觉，加大的车灯和丰田标志更为醒目。在载物能力上，国产RAV4也有着不俗的表现，通过将后排座椅放倒，最终释放出的储物空间有1031升，大件物品的存放将不再成为问题。由于RAV4的备胎被挂置在了尾门上.这就为车内腾出了更多可搁放物品的空间，在行李厢的地板下方，隔出的储物格方便了车主存放日常用品。
通过以上近30款车目前的库存量以及优惠情况的调查我们不难发现，新车型：如科鲁兹、新君威以及时下热门的SUV车型：奥迪Q5、大众途观等几乎是“0”库存状态。而已经上市多时，且市场保有量较大的车型如福克斯、卡罗拉、比亚迪F0等车型的库存量较大，在价格上也有一定的优惠。
