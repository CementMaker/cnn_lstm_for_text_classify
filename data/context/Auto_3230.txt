不容小觑 试驾海马汽车首款SUV--骑士
操控篇：底盘扎实，车身紧绷。
 
这两天的试驾，体验了城市，高速，国道，县道，山区弯道，颠簸路等各种路况，对车辆有了比较全面的体会。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

骑士采用了难得的硼钢材质更添加了车身的强度，3H高刚度车身，车身紧绷，有着不错的抗扭曲能力，也无任何异响。行进在高速公路中，感觉骑士的悬架和减震调教比较硬朗，高速过弯时，底盘很整，侧向支撑不错，给驾驶者以信心。
根据工信部油耗，骑士的百公里综合油耗为8.8升，对于2.0排量，超过1.4吨整备质量来说，还是比较省油的。或许是出于燃油经济性的考虑，骑士的动力调教不那么激进，油门响应较慢。感觉发动机在低转速范围内动力比较好，中高转速加速不够线性。
		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

骑士的助力转向系统手感轻盈，油门，离合和刹车力度适中，女性朋友一样可以轻松驾驭，用户群体得到了进一步的扩展，不过，高速时骑士的转向调教的更沉一些或许能增加过弯并线时的稳定性。方向盘的调教兼顾了越野路况，行车在颠簸烂路，方向盘随路况的高低左右变化中不至于打手；当然在公路行驶时，方向盘对驾驶者指令的反馈不那么直接，这也是综合各种路况的折中方案，无可厚非。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

海马骑士先期上市的都采用的是5速手动变速箱，档位清晰，换挡行程适中，入挡顺畅，换挡力度也比较适中。喜欢高坐姿的驾驶者把座椅调到最高时，换挡手柄位置稍低，个人举得需要短暂的适应。驾驶座椅和方向盘调到最高时，方向盘上沿遮挡了一些仪表盘。
