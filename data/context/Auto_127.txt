国产新普拉多车展首发 进口2.7L加价3万
网上车市4月16日报道
今年1月初，丰田全新一代进口PRADO普拉多2.7L在中国正式上市，共推出了三款车型，售价为43.7万元——49.9万元。而随着国产新普拉多现身新车目录，4.0升车型还是会交由一汽四川丰田生产。另外笔者致电一汽丰田客服中心了解到，国产新普拉多4.0L车型将在此次北京车展亮相。笔者又通过一汽丰田4S店了解到，老款国产的4.0L车型目前已经停产，只有个别经销商处有少量库存车，并且在售的新款2.7L车型店内有少量现车供应。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 



据一汽丰田4S店销售人员介绍，老款国产普拉多目前已停止销售，只有个别经销店还有少量库存车，但在近期也会全面售清。现在在售的普拉多为进口新款2.7L车型，店内有少量白色、绿色现车可供消费者选择，但提现车根据车型需要加价2-3万元。另据介绍，新款国产4.0L也要即将面世，但价格方面要高于老款车型，详细的上市时间还暂未公布，不过在此次北京车展期间我们可以近距离接触到这款新车。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

即将推出的国产新普拉多4.0L车型在动力方面采用代号1GR-FE的3955ccV6发动机，可以在5600rpm发出203千瓦的最大功率，在4400rpm发出380Nm的最大扭矩。行车方面提供四种模式，分别为泥沙模式、碎石模式、沟壑模式及石块路面模式，可以实现加力档、4驱/后驱的切换等功能。此外，新普拉多可以对前后制动力进行40：60的分配。4.0L的版本还配备有能对车辆下坡速度进行限制的陡坡缓降系统。该车的接近角、离去角、轴间通过角分别为32度、28度、22度。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

全新一代PRADO普拉多在外形上较上一代车型有了显著变化，整体风格更趋大气时尚。前保险杠、前大灯、前雾灯经过了全新的设计，更显尊崇。前格栅的设计更加突出厚重感，车身线条和整体轮廓更具现代美感。新车的尺寸为4,820mm×1,885mm×1,845mm（长×宽×高），轴距为2,790mm，更好地实现了空间利用率和驾乘舒适性。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

在内饰上，全新一代PRADO普拉多更加高档精致、用料也更为考究，整体风格采用了很多豪华元素以及对称式设计，直立式中控台以及合理的按键布局让车内控制系统更具便捷性，手动四方向可调式真皮方向盘和倒车雷达以及自动豪华版配有的倒车影像监视系统为驾驶者提供了更加人性化及便利的装备。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

目前在售的新一代普拉多配备了2.7L直列四缸发动机，应用了顶置双凸轮轴VVT-i技术，最大功率120kW/5,200rpm，峰值扭矩246Nm/3,800rpm，与其相匹配的是经典的4挡自动变速箱和5挡手动变速箱，动力充沛、操控精准。与很多高端SUV采用的承载式车身结构不同，全新普拉多仍保留了非承载式车身结构，并通过进一步优化，提升了车体的扭转刚度和弯曲刚度。新普拉多还配备可以便于从泥淖中脱困的“极低速走行模式”、KDSS侧倾控制系统；AVSS车辆升降系统能够在高速下使车辆降低20mm，而在恶劣路况下可以提高40mm。此外，新车还采用了先进的前双叉独立式、后四连杆机构式悬架结构，标配了265/65R17大尺寸轮胎，完美地兼顾了越野道路的通过性和城市道路的舒适性。
编辑点评：丰田旗下的兰德酷路泽及普拉多都是擅长越野的利器，并且有着不错的口碑。而当新普拉多上市后我们发现并没有大排量车型时，让许多爱好越野的消费者转而投向其它品牌或处于待购的状态。此次新款4.0L的推出，势必会让许多消费者着迷，目前进口2.7L车型提现车需加价3万元，相信国产4.0L推出时也会如此。但普拉多的持续加价相信对喜爱这款车的消费者影响并不是很大。
