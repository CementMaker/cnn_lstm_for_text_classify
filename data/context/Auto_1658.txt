最实用的MINI  Countryman最快年底进口
汽车之家4月30日报道
 在今年北京车展的宝马展台上，全新一代5系加长版和改款X5的亮相似乎让另一端MINI的展台显得有些失色，但是我们不能否认MINI的全新产品Countryman是一款非常重要的全新产品，而这款全新产品最快将于今年底正式进入中国市场销售。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

作为目前MINI产品系列中尺寸最大的车型，Countryman车长达到4.1米，但是外观依旧不改可爱的设计主调，在个性需求上依旧不落后其它产品，在丰富了MINI产品线的同时，也让我们看到了其品牌产品实用性的一面。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

虽然动力方面仍旧配备的是1.6升汽油、柴油发动机，不过值得一提的是Cooper S汽油机首次增加了双涡管单涡轮增压器和燃油直喷技术，最大功率达135千瓦。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

在内饰部分，这款全新产品依旧体现了其个性的一面，Countryman采用标准的5门掀背式设计，配备4个独立座椅，后排也可选装3尾座，双尾座的中间配备了两个杯架及滑轨，其目的是为用户提供一个较为灵活的储物空间。其他的配置还包括全景天窗、自适头灯、加热挡风玻璃、导航、蓝牙以及经过升级的音响等。
