兼具优雅与动感 奥迪A5/S5湖南车展正式上市
汽车之家5月28日报道
5月27日，是2010长沙车展开幕的第一天，上午10点，刚刚在全国范围上市的奥迪A5/S5系列轿跑车正式登陆湖南，亮相长沙车展，今后奥迪A5/S5将正式进入湖南市场，成为一汽-大众奥迪湖南经销商湖南华运通4S店和湖南华洋华迪4S店内的新成员。

奥迪A5是奥迪百年历史上设计最出色的车型之一。在世界汽车史上，从未有一款车像奥迪 A5 那样，如此完美地将优雅与动感融为一体。奥迪 A5创领了汽车设计“优雅运动”的新潮，也将因此而成为世界汽车设计史上的一个经典之作。 此次上市的 A5/S5 系列车型阵容强大， 包括全新上市的 A5/S5 Cabriolet 敞篷轿跑车、 A5/S5 Sportback 五门轿跑车和经过全新升级的 A5/S5 Coupé 双门轿跑车，共 7 款车型。A5/S5 系列轿跑车根据不同配置分为 10 个版本，价格区间为 49.39 万元——99.8 万元。其中，为了满足国内用户的个性化需求，奥迪在 A5 Coupé 2.0T基础上推出了充满动感个性的 A5 Coupé 2.0T风尚版。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

奥迪 A5 没有使用夸张的肌肉和锋利的线条来营造动感和力量，而是以短前悬、宽车身、低底盘、收紧的尾部和干净利落的轮廓线，赋予它一种与生俱来的运动气质和蓄势待发的力量感。最为经典的是车身侧面那道极富动感而又毫不夸张的波浪形腰线，使奥迪 A5 在动感中透出一种无与伦比的优雅与精致。此外，无论从哪个角度看，奥迪 A5 各部分的比例都极为协调、流畅，呈现出一种精妙而大气的和谐之美。这种对动感与优雅创造性的完美融合，将世界汽车设计美学提高到一个全新的境界，为奥迪A5 赢得了被称为“设计界的奥斯卡”的德国联邦设计大奖、作为汽车界最重要奖项的“金方向盘奖”等国际权威设计奖项，以及国际权威媒体授予的多项设计大奖。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 |

作为奥迪 A5 系列推出的首款车型，出自前奥迪著名设计大师瓦尔特&#8226;德&#8226;西尔瓦之手的奥迪 A5 Coupé 双门轿跑车，以独树一帜的“优雅运动”为设计理念，为 A5 系列车型奠定了具有非凡魅力的设计基调。 奥迪 A5 Cabriolet 敞篷轿跑车延续了奥迪对经典敞篷车型的定义，采用高品质的软质车顶，不仅拥有同级别车型中最快的车顶开启/关闭速度，在车顶开启状态下行李厢容积达到同级别最大的 320 升，将优雅的造型设计与实用性完美融合。

 |

奥迪 A5 Sportback 五门轿跑车展现了奥迪设计师的创新智慧，继“Avant”旅行轿车之后，奥迪以“Sportback”这一新的设计概念，将个性驾驶与多功能性融为一体，从而再次引领了潮流，并在高档中型轿跑车中开辟了全新的细分市场。奥迪 A5 系列车型配备了 2.0 TFSI 、3.2FSI 两款高效发动机，除了奥迪领先的涡轮增压和燃油直喷技术外，这两款发动机均采用了独创的 AVS 奥迪可变气门升程系统，在提供充沛动力的同时兼顾杰出的燃油经济性。
作为 A 系列轿车的高性能版本，奥迪 S 系列车型不仅拥有更为动感的外型设计，其强悍的运动性能更充分满足了驾驶者的驾驶激情。高性能版的奥迪 S5 系列车型配备了两款动力更为强劲的 4.2 FSI 和 3.0 TFSI 发动机。其中，全新上市的 S5 Cabriolet 敞篷轿跑车和 S5 Sportback奥迪 A5 Sportback 奥迪 A5 Cabriolet 奥迪 A5 Coupé五门轿跑车搭载的 3.0 升 TFSI 发动机代表了奥迪最前沿的领先技术，机械增压与燃油直喷技术的完美结合迸发出强大的动力性能。245 千瓦的最大功率和 440 牛&#8226;米的最大扭矩堪与 V 型 8缸发动机相媲美。值得一提的是，这两款高性能车型都采用了奥迪最为领先的传动系统——7 速S tronic 双离合变速箱和 quattro 全时四轮驱动系统，使驾驶者充分体验充满激情而高效的驾驶感受。
此外，全新上市的 A5/S5 系列轿跑车的一大亮点是全系车型配备了能量回收系统。 该套系统能在汽车减速或巡航过程中，将多余的车辆动能将转换为电力并回收至电力系统。而在加速过程中，这些电能将用来供给车上的电装系统，从而减少发电机组对于引擎效能的消耗，使车辆具备了更为出色的高效性能。
作为奥迪最新一代高档中型轿跑车， 奥迪 A5/S5 系列车型不仅具备优雅的设计和出色的运动性能，其丰富的豪华配置同样处于同级别领先水平。第三代 MMI 多媒体交互系统不仅功能强大，出色的人机工程学设计为驾乘者提供了惬意的汽车娱乐生活。丹麦顶级 Bang &amp; Olufsen 音响系统可演奏出宛如歌剧院般的完美音乐效果，使车内乘员得以在优雅的环境中畅享驾驶的乐趣。此外，奥迪 A5 系列车型还提供丰富的选装配置，消费者可根据自己的个性化需求对车辆的外观、内饰和配置进行自由搭配，打造彰显个性风格的魅力座驾。
