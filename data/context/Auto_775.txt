不变的"加长效应" 图解新一代奥迪A8L
动力方面，除沿用A8的4.2 FSI V8、4.2 TDI V8、3.0 TFSIV6以及3.0 TDI发动机外，作为豪华车的地位之作，A8L将继续推出W12产品。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

 
全新的6.3L FSI W12发动机能够爆发出368千瓦（500马力）的最大功率，能够在3250转时输出625牛米的峰值扭矩。配合一台全新的8速变速器，A8L能够在4.9秒内完成百公里加速，而且能轻松达到250公里/小时的限定车速。
这台W12发动机可以看作是两台气缸夹角很窄（15度）的V6发动机的合体。两台V6发动机之间的夹角为72度，这种布局使得W12体型十分紧凑，长度刚好超过50厘米，甚至比普通的V8引擎还要短！
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

 
相对于旧款6.0L W12发动机，新发动机扩大了缸径，在FSI缸内直喷技术的帮助下，新车功率大幅增加37千瓦，而且二氧化碳排放还比之前降低12%。
与新A8一致，A8L的底盘进行了更新，轻质铝金属底盘机构让车身重量较采用钢质车身的车型降低了40%左右。
奥迪新A8L的Quattro四驱也需要叙述一下，为了获得更好的驾驶操控感，在默认模式下，通过中央差速器，奥迪新A8L的60%动力都会分配到后轴，这也符合目前运动化车型的设定，但是在极限状态下，新A8可以后轴或者前轴全部承载100%的驱动力。在大部分情况下，只要有一侧车轮还有附着力，那么新A8L就应该可以脱困，从而避免空转的产生。这些原理的诞生主要因为奥迪A8L采用了最新的扭矩矢量分配系统（SportDifferential）。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

 
在四驱系统的帮助下，操控性大幅度提升，前后带副车架的独立悬架带来了精准的操控性，通过电控的转向伺服机构，新A8L可以在不同的车速和驾驶环境下拥有更适应当前状态的路感、转向比。
作为奥迪旗舰车型，A8L在科技配置方面更是下足心思。新一代的MMI系统内容更多，功能增强，而且控制也更简化了一些。驾驶员信息系统（DIS）采用了新的显示方式，在仪表板时速与转速表的中间区域采用全新的多媒体显示界面，这种设计更加合理。

作为中国豪华车市场的耕耘者，奥迪对中国市场有相当深入的了解。旗下A4/A6等车型纷纷应消费者需求，推出了加长版本，在细分市场中，也取得了领先成绩。不过在更高一个级别，上一代A8却没有取得好的成绩，宝马7、奔驰S的地位依然牢固。顺延加长之风，A8L对国内D级车市场再次发起冲击，较以往不同，此次似乎来得凶猛许多。
