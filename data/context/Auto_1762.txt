新车盘点 上市新车多需预订
“现在能有不用排队等交车的新车吗？”在一家4S店，记者听到一位消费者正对销售人员抱怨。从去年以来，每周都会有新款或换代改款的车型上市，尤其上个月上市的新车多达数十款，然而大多数车型都只能看而不能买，“只有样车”，“只能接受预订”，“没有任何优惠”，因此导致一部分消费者暂缓买车计划，持币待购等待市场价格出现优惠。
明锐
新明锐预订需等1个月

 |

 | 

新款明锐于4月中旬上市后，经销商处只有小批量现车到店，市场价格没有出现任何优惠，而老款明锐因此前优惠促销力度加大，目前已经基本销售一空。
斯柯达经销商介绍，近期到店的新款明锐以银色和灰色为主，其他颜色需要预订，订金2000元，等待周期为4到6周。
ix35
挤压途胜让利2万
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

ix35是北京现代上市的新款SUV，与途胜平行销售，然而两款定位相近的新老车型一同销售，其结果就是挤压老款车型的市场份额，目前途胜最高优惠达到2万元。
北京现代经销商介绍，ix35现车数量不多，其中以银色的2.0L自动挡车型为主，其他车型均需要预订，订金为1万元，提车周期为3至15个工作日。
荣威350
只有展车到店
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

刚刚公布价格的荣威350现在已经有展车到达经销商处，已接受预订，订金为2000元，提车时间尚不能确定。
荣威经销商介绍，目前店内的展车为350D讯豪版，银色。现在只能接受预订，但具体交车时间暂不确定，需要根据订单及厂家排期来进行提车。
睿翼轿跑
开始分批消化订单
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

睿翼轿跑上市已经近一个月，此前在经销商处预订的订单开始分批交车，据了解，首批睿翼轿跑车型会在近期交付车主。
一汽马自达经销商介绍，睿翼轿跑目前有黑色2.5L至尊版展车，但是仍没有现车供应，消费者需要预订等待，订金为5000元，等待周期为1至2个月。
北斗星e+
新款尚未到店
在北京车展上市的北斗星e+是昌河铃木推出的全新改款车型，由于刚刚宣布上市，近期还没有现车到达经销商处，据了解，该车在5月底前可以开始销售。
昌河铃木经销商介绍，北斗星e+目前可以接受预订，订金为1000元，最快5月底可以提车。同时09款北斗星有现车销售，而且有小幅优惠，可以送2.5%购置税和200元加油卡。
