由历史说起 新老代同时销售车型
汽车之家4月20日报道
之前已经介绍过两期新老款同时销售的车型，但这个话题依然没有说完，LANCER可谓是三菱的当家车型之一，在WRC战场上的立下的丰功伟绩也是众人皆知，它也同样拥有着不算短的历史，而韩国现代的索纳塔和伊兰特都是在全球市场的重要车型， 它们在中国都有着二代同堂的现象存在。
我们固然希望国内的合资厂商能够同步换代、与时俱进，对于新老共存的现象也不能认定地很偏激，毕竟老款产品降价使消费者得到了更多的实用产品，其实这种现象并非只在国内存在，在其他一些国家也同样存在，也算是给那些“愤青”们一些安慰吧。

第一代LANCER
『第一代LANCER的两门款车型』	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

第一代LANCER是又三菱Colt Galant车型演变而来，于1973年面世，定位在Minica和Ganlant（戈蓝）之间，优美的外形符合空气动力学的设计，在性能上也表现不俗，它在同年参加了澳洲SouthernCross Rally拉力赛并获得了全面的胜利，第二年参加了第22届Safari越野赛和肯亚越野赛获得综合优胜，因此第一代LANCER也与越野拉力赛结下了不解之缘。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

第一代LANCER有四门版和三门版，采用4G42发动机，有1.2升、1.4升和1.6升三种排量，其中双门版的1.6升的GSR车型还采用了双化油器，最大功率达到110马力，配合仅825公斤的车重，因此动力性能显得非常突出。
第二代LANCER
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

6年之后的1979年，第二代LANCER上市了，采用新的反向平衡轴设计，最大程度减轻发动机工作时的震动，而且在当时已经采用了四连杆的独立悬挂，不仅增强了舒适性，而且为其后优秀的操控性打下了基础。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

到1980年，LANCER共有五种动力可供选择，1.2L、1.4L、1.6L、1.8L和1.8T，这也是第一台采用涡轮增压发动机且采用后轮驱动的LANCER车型，1981年，三菱为这台涡轮增压发动机加装了中冷器，最大功率达到165马力，也受到广大车迷和赛手们的欢迎，其1.6 GSR车型也受到了不少香港车迷的追捧。据说正是因为性能好速度快，LANCER经常被不法之徒用作逃跑的工具。
第三代LANCER
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

1982年，以三菱Mirage为基础设计的第三代LANCER发布，它与第二代的后期车型外形差别不大，在海外市场称为LANCER Fiore，而Mirage车型作为五门掀背款同时销售，在澳大利亚市场，外表经过一些修改，最终作为三菱Colt车型进行销售。
第四代LANCER
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

在接下来的1983年，三菱推出了第四代LANCER车型，同时Mirage和LANCER也成为了同一款车型，它的外观向时尚化靠拢，采用电脑计算的1.6升涡轮增压电喷发动机，最大功率为120马力，三菱的1.8升柴油和1.5升MD (Modulated Displacement) 也是首次亮相。1985年，LANCER旅行版的亮相丰富了产品线，一年后，全时四驱版本的LANCER随之发布，这款车型也受到海外和日本本土的广泛追捧，无论是在商务还是家用市场都取得了良好的销量。
第五代LANCER
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

1988年，第五代LANCER上市，它的紧随当时戈蓝的外形设计，外表变得动感流畅起来，流线的设计营造了出色的空气动力学，这代车型在内饰和乘坐空间方面也有很大提升，五门掀背款车型加入到LANCER的阵营中。在澳大利亚，所有车型都叫做LANCER，而在日本本土则叫做Mirage Aspire。同样地，全时四驱技术也被应用于第五代LANCER车型。
第六代
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

1991年，第六代LANCER面市，这带车型充满着浓厚的拉力赛气息，ABS、电动座椅、四轮碟刹的配置方面在当时非常先进，全新设计的1.6升16气门发动机在同级车十分领先，同时三菱还打造了一款具有最细汽缸容积的1.6升V6发动机，可谓世界首创，MIVEC（智能可变气门正时与升程控制系统）也是第一次应用于LANCER的发动机，倍受车迷追捧的LANCER EVO（Evolution）车型就是从这代开始的，第六代LANCER在澳大利亚、菲律宾、欧洲和拉丁美洲都取得了不错的市场成绩。
● 
东南菱帅就是以第六代三菱LANCER为原型生产的。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

由三菱注资的台湾中华汽车引进了日本三菱的第六代LANCER车型，进行本土化改造后，在台湾地区获得了非常大的销量，于是在2003年，大陆的东南汽车引进了这款台湾版的三菱LANCER，并起名为Lioncel（菱帅）。
	
	
		
		
											   		          				
 | 
	    
 | 
	    
 | 
		
 | 
		
 | 

经过东南汽车几年的发展和实力的增强，如今的“菱帅”已经成为具有自主知识产权的“V3菱悦”，外观上依然保留着原型车LANCER的特点，只是在前后灯和车身细节方面进行了一些小修饰，比菱帅更加有精神，在内饰方面V3菱悦也更加年轻圆润。
作为家用轿车的V3菱悦，2500mm的轴距在同级车中处于劣势，因此也直接造型的较拥挤的后排空间，动力系统采用排量为1.5L的直列四缸发动机，带可变气门正时技术，最大功率为120马力，可搭配5挡手动或带6速模拟挡位的CVT无级变速箱。V3菱悦的全系车型都在十万以下，最低配置的1.5L启航版的售价为5.98万，该车型整体价位还是比较具有吸引力的。
